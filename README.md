# 💻 My Personal Portfolio Website

Welcome to the code repository for my personal website! I built this to showcase my projects, skills, and experience in one place. It will be ever evolving as I gain new skills and learn new things.  

## 🔗 Live Link
Check out the website here: **[drew-kirby.com](https://drew-kirby.com)**

---

## 🛠️ How it was built
This website is built from scratch using clean, standard web languages:
*   **HTML5** for the core structure
*   **CSS3** for layout styling and mobile responsiveness
*   **JavaScript** for interactive features and logic

It is hosted for free using **GitHub Pages** and linked to my custom domain.

---

## 📂 Project Files
*   `index.html` - The main homepage of the site
*   `/assets` - Images, icons, and styling files
*   `README.md` - You're reading this right now!

---

## 🚀 Running it locally
If you want to view the code on your own computer:
1. Download or clone this repository.
2. Open the `index.html` file in any modern web browser.
